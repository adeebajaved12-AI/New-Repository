# CodeAlpha_Language_Translation_Tool
**Internship Project - Task 1**

## Project Description
A professional AI-powered Language Translation Tool developed using Python. This application allows users to translate text between multiple languages (e.g., English to Urdu, Arabic, French) with a clean and interactive web interface.

## Features
- **Modern UI:** Built with Gradio for a sleek user experience.
- **API Integration:** Utilizes the Google Translate API (via `deep-translator`) for accurate, real-time results.
- **Auto-Detection:** Automatically identifies the source language of the input text.
- **One-Click Copy:** Includes a built-in copy button to easily save the translated output.
- **Public Hosting:** Can be accessed via a temporary public URL provided by Gradio.

## Tech Stack
- **Language:** Python
- **Framework:** Gradio
- **API:** Google Translate API (deep-translator)
- **Environment:** Google Colab

## How to Run
1. Open the provided `.ipynb` file in Google Colab.
2. Run the first cell to install dependencies: `pip install gradio deep-translator`.
3. Execute the logic and UI cells.
4. Click on the generated **Public URL** to open the app in your browser.
